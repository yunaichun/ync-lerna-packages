## 简介

React 组件 npm 包模版。

## 如何开发

```bash
# 安装依赖
npm ci

# eslint 规范
npm run lint

# test 测试用例
npm run test

# 发布前构建
npm run build

# 发布
npm publish
```

## 如何使用

``` bash
# 安装:  
npm i react-component --save

# 引用
import Index from "react-component"

# 调用
<Index />
```
